## Assignment - Typed

```
let a: int = 5;
```

## Assignment - Shorthand, Implicit Type

```
a := 5
```

## If statement:

```
if a > b {
    print("A is greater");
}

else {
    print("B is greater");
}
```

## Print (Standard Lib)

```
print("Foo");
print("Foo is ", foo);
print("Foo is ) + foo);
```

## Functions:

```
fn greater_than(arg1: int, arg2: int) -> bool {
    if arg1 > arg2 {
        return true;
    }
    else {
        return false;
    }
}
```
