# Types

## Note: Reading this document:

Here is the basic structure (sample) for each data type which is represented in this document.

```
Data Type Name
Notation -> Dentotes how it's represented in fir
C        -> Corresponding data type in the C programming language
Memory   -> Memory requirement (in bytes)
```

### 1. Integer

- Notation : int
- C : int
- Memory : 4 bytes

### 2. Character
- Notation : char
- C : unsigned/signed char
- Memory : 2 bytes

### 3. String
- Notation : string
- C : char[]
- Memory : 8 bytes

### 4. Boolean
- Notation: bool
- C : bool
- Memory : 1 bit

### 5. Float
- Notation : float
- C : float/double
- Memory : 4/8 bytes
